# C++ Primer学习笔记之重载运算与类型转换

关于重载运算符值得注意的几点：

+ 运算对象求值顺序的规则无法应用到重载的运算符上
+ 重载的&&和||运算符无法保留短路求值属性

通常情况下，**不应该重载逗号、取址、逻辑与和逻辑或运算符**。
